GameMage - User Guide

Introduction

Welcome to GameMage, your personal game catalog management application. This guide will help you understand how to use all the features available in the application.

Window Controls

Minimize (_) Minimizes the application window to the taskbar.
Maximize/Restore ([]) Toggles between maximized and windowed mode.
Close (X) Closes the application.

Main Menu

File Menu for importing and exporting game data.
  • Import JSON File - Load a catalog of games from a JSON file.
  • Export JSON File - Save your catalog of games to a JSON file.
Help Opens this user guide.

Navigation

All Shows all games in your catalog.
Favorites Shows only games you've marked as favorites.

Sorting and Filtering

Sort By Changes how games are ordered in the view.
  • Title A-Z - Sort alphabetically by title.
  • Title Z-A - Sort reverse alphabetically by title.
  • Release Year⬆ - Sort by year, oldest first.
  • Release Year⬇ - Sort by year, newest first.

Filters

The right side panel contains various filters to narrow down your game list:

Search Find games by title.
Genre Filter games by genre.
Platform Filter games by platform.
Tags Filter games by tags.
Release Year Filter games by their release year.
Publisher Filter games by publisher.
Developer Filter games by developer.

Game Management

Add Game Opens a form to add a new game to your catalog.

Game Cards

Each game in your catalog is displayed as a card with the following options:

  • Edit - Modify the game details.
  • Delete - Remove the game from your catalog.
  • Favorite - Mark or unmark a game as favorite.

Pagination

Navigate through multiple pages of games using the pagination controls at the bottom:

First Page (<<) Go to the first page of results.
Previous Page (<) Go to the previous page.
Page Number Shows the current page number.
Next Page (>) Go to the next page.
Last Page (>>) Go to the last page of results.

Tips & Tricks

Tip: You can combine multiple filters to narrow down your search results.

Tip: Use the search function for quick access to specific games.

Tip: Export your catalog regularly to create backups of your game library.
